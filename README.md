# podcast-app-client

A podcast app built with Vue.Js, Vuex and backed up by a slimPHP API + PostgreSQL Database.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
